51万年历API文档
使用说明
51万年历接口提供专业全面的传统黄历信息；及时更新的法定调休假安排；大陆、港澳台四地的节日节气信息。
申请试用authkey请联系:
商务总监龚小姐（QQ：6635 2806）或产品总监李先生（QQ：7043 4724 3）
通用参数说明
token: 根据请求参数(加上authorisekey 在群组中给出)字典排序拼接字符串 然后md5 后的值
type: 0 表示大陆 1表示香港 2表示台湾 3表示澳门
client: 请求的客户端来源
获取调休假
demo链接： http://co-api.51wnl.com/calendar/vacations?token=A2E0C3CDEA081D3BFC34F8FE23A15886&type=1×tamp=1462377600&client=ceshi
参数说明：
token
type
client
返回值说明:{"status":0,"data":{"20140101":"1","20140131":"1"},"errMsg":""}
data:字段为主要输出内容 对应的值表示 时间以及当天的上班状态 1表示休假 2表示上班
获取节日
demo链接： http://co-api.51wnl.com/calendar/festival?token=A2E0C3CDEA081D3BFC34F8FE23A15886&type=1×tamp=1462377600&client=ceshi
参数说明：
token
type
client
返回值说明:
R：上次修改时间
S：按阳历计算的节日
L：按农历计算的节日
W：按周计算的节日
"0511"：S,L时表示5月11日 W时表示5月的第一个星期一
